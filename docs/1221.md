**质子框架**是一个类似于其他渗透测试框架的 Windows 后开发框架。主要区别在于，它使用 Windows Script Host 执行大多数操作，其核心兼容性支持默认安装 Windows 2000，从 Windows 10 一直到 Windows 2000 都没有服务包。

![](img//c42991e4fd62ee58db8d8d7780777dd6.png)
