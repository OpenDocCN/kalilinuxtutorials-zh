**PowerGram** 是一个纯 PowerShell 电报机器人，可以运行在 Windows、Linux 或 Mac OS 上。要使用它，您只需要 PowerShell 4 或更高版本以及互联网连接。

Bot 和 Telegram 服务器之间的所有通信都是用 HTTPS 加密的，但是所有请求都是用 GET 方法发送的，所以很容易被拦截。

# 要求

*   PowerShell 4.0 或更高版本
