AzureGraph 是微软 Graph 上的 Azure 广告信息收集工具。

得益于微软 Graph 技术，可以从 Azure AD 获取各种信息，比如用户、设备、应用、域等等。

这个应用程序允许您通过 PowerShell 控制台以一种简单易行的方式通过 API 查询这些数据。此外，你可以从云中下载所有信息，并完全离线使用。

## 要求

*   PowerShell 4.0 或更高版本

## [计] 下载
