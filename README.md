## 下载

### Docker

```
docker pull apachecn0/kalilinuxtutorials-zh
docker run -tid -p <port>:80 apachecn0/kalilinuxtutorials-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g kalilinuxtutorials-zh
kalilinuxtutorials-zh <port>
# 访问 http://localhost:{port} 查看文档
```
