Cerbrutus 是一个用 Python 编写的模块化暴力工具，用于非常快速的密码喷射 SSH、FTP 以及不久的将来的其他网络服务。

即将推出: **SMB、HTTP(s) POST、HTTP(s) GET、HTTP BASIC AUTH** *感谢@0dayctf、Rondons、Enigma、001 的测试和贡献*
