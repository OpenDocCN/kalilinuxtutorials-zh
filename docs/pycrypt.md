## **如何使用？**

*   在 GitHub 上找到任何基于 [Python](https://kalilinuxtutorials.com/bayanay/) 的后门/RAT。
*   用 pycrypt 加密它的负载
*   现在使用 pyinstaller 将加密的有效负载转换为 exe

## **如何运行**

### ***:-对于 Windows:-**

确保安装了 python3 和 pip，并且还安装了需求

***python pycrypt.py***
