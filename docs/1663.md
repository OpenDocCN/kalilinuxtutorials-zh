**Sak1to-Shell** 是用纯 C (Windows)编写的多线程 c2 服务器和反向 TCP shell 客户端。

**命令列表**

**列表:**列出可用的连接。
**交互【id】:**与客户端交互。
**下载【文件名】:**从客户端下载文件。
**上传【文件名】:**上传文件到客户端。
**后台:**后台客户端。
**退出:**终止客户端或服务器。
**cd [dir]:** 更改客户端上的目录
