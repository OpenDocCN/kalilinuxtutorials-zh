框架 **RapidPayload** 是一个 metasploit 有效负载生成器。

**要求**

*   OpenJDK 8 (JAVA)，或更高版本。
*   Metasploit
*   Apktool
*   Python3

**执行**

git 克隆[https://github.com/AngelSecurityTeam/RapidPayload](https://github.com/AngelSecurityTeam/RapidPayload)
CD rapid payload
bash install . sh
python 3 rapid payload . py
