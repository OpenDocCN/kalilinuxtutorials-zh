**SQL 注入**是最广为人知的针对网络应用的攻击之一。以下是 2019 最佳 SQL 注入工具列表。

它的攻击包括通过从客户到应用程序的信息插入或“注入”SQL 查询。

有效的 SQL 注入努力可以从数据库中读取微妙的信息，改变数据库信息(插入/更新/删除)，在数据库上执行组织任务(例如，关闭 DBMS)，在 DBMS 记录框架上补偿给定文档显示的实质，并不时地向工作的操作系统发出命令。
