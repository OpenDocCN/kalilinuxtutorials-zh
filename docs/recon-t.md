Recon-T 专为侦察和信息收集设计，强调简单性。它将做从。

**特性**

*   信息安全标题
*   晶片检测器
*   抢横幅
    *   电话号码
    *   信用卡号码
    *   电子邮件
    *   美国社会安全号码
*   Url 爬网
    *   dom 参数 url
    *   内部动态参数
    *   外部动态参数
    *   内部链接
    *   外部连接
*   端口扫描仪
*   子域枚举

**Recon-T** **需求**

*   点击
*   要求
*   色彩日志
*   bs4
*   右旋糖酐
