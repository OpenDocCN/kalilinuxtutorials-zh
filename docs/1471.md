*   该工具必须以 ROOT 用户身份运行！！！

git 克隆 https://github.com/Z4nzu/hackingtool.git
chmod-R 755 hacking tool
CD hacking tool
sudo pip 3 install-R requirement . txt
bash install . sh
sudo hacking tool
