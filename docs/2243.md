**Gotanda** 是针对 Firefox/Chrome 的 OSINT(开源智能)Web 扩展。

这个 Web 扩展可以在网页中搜索来自某个 IOC 的信息。(IP、域、URL、社交网络等)

这个资源库部分用于学习和 JavaScript 实践。

下载下面的链接。

*   火狐浏览器
*   铬

**用法**

右键单击突出显示的 IOC 字符串，将显示上下文菜单。(或者右键单击任何链接。)

当你想使用某个引擎进行搜索时，你可以从列表中选择一个。
