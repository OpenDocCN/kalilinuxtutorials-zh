**FakeDataGen** 是一个完全有效的伪数据生成器。这个工具帮助你用完全有效的数据创建假账户(西班牙格式)。在这些信息中，您可以找到最常见的姓名、电子邮件、银行详细信息和其他有用的信息。

**要求**

*   python3
*   安装要求. txt

**下载**

建议克隆完整的存储库或下载 zip 文件。您可以通过运行以下命令来实现这一点:

```
git clone https://github.com/JoelGMSec/FakeDataGen
```

**用途**

```
./FakeDataGen.py -h
```
