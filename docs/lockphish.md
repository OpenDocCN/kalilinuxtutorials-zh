**Lockphish** 这是第一款针对锁屏钓鱼攻击的工具(2020 年 5 月 13 日)，旨在通过 https 链接获取 Windows 凭据、Android PIN 和 iPhone 密码。

**特性**

*   Windows、Android 和 iPhone 的锁屏钓鱼页面
*   自动检测设备
*   通过 Ngrok 进行端口转发
*   IP 跟踪器
