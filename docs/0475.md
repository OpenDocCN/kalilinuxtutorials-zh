# Xerxes:增强了许多特性的 Dos 工具

> 原文：<https://kalilinuxtutorials.com/xerxes-dos-tool/>

Xerxes dos 工具增强了许多压力测试特性。它有许多功能，其中一些功能是:

*   TLS 支持
*   HTTP 标头随机化
*   用户代理随机化
*   多重处理支持
*   多种攻击媒介
*   等等…

不仅如此，我们还在积极开发它，增加更多的特性和功能。
