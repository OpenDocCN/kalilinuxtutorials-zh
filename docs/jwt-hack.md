**JWT 黑客**是对 JWT 进行黑客攻击/安全测试的工具。支持编码/解码 JWT，为 JWT 攻击和快速破解(dict/brutefoce)生成有效载荷。

**安装**

*   **go-get(开发版)**

**$ go get-u github.com/hahwul/jwt-hack**

*   **自制**

**$ brew tap 哈哈哈/jwt-hack
$ brew install jwt-hack**

*   **snapcraft**

**$ sudo snap 安装 jwt-hack**
