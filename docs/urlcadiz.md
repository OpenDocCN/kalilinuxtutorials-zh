一个简单的脚本来为社会工程生成一个隐藏的 url。

**安装(在 Kali Linux 2020.2 上测试):**

**git 克隆 https://github.com/PerezMascato/URLCADIZ
sudo pip 3 安装 pyshorteners
CD URLCADIZ
python 3 URLCADIZ . py**
