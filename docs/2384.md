**EDRHunt** 扫描 Windows 服务、驱动程序、进程和注册表，寻找已安装的 edr(端点检测和响应)。阅读更多关于 EDRHunt 的信息

## 安装

*   二进制的
    *   从版本部分下载最新版本。版本是为 windows/amd64 构建的。
*   去
    *   需要在系统上安装 Go。在 Go1.17+上测试。
    *   `**go install github.com/FourCoreLabs/EDRHunt/cmd/EDRHunt@master**`
